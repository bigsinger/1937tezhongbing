# 《1937特种兵：敌后武工队》Win10/Win11 兼容补丁 v1.1.0

发布日期：2026-07-18

## 这个补丁解决什么问题

本补丁用于让《1937特种兵：敌后武工队》绿色版在 Windows 10/11 上流畅、稳定运行，主要解决：

- 启动时报 `DirectDraw Error DDERR_UNSUPPORTED`；
- 弹出 `Can't Load Game Files Library!` 误报；
- 片头、菜单、任务简报或点击“开始游戏”后长时间“未响应”；
- 进入关卡后帧率低、旧循环占用 CPU 或偶发微卡；
- 全屏不能正确采用当前 Windows 桌面分辨率。

最终方案使用官方 cnc-ddraw 7.1.0.0 将 DirectDraw 转为 Direct3D 9，并附加本游戏专用 DirectInput 兼容层。原版 `M1937.exe` 不会被改写。

## 适用版本

安装器只接受经过完整测试的绿色版：

```text
文件：M1937.exe
SHA-256：F4DD1131DF6C993C01EA011F9439BC725E6DC6491B5FBBA47724D7D5B64DA3F3
```

其他版本的程序布局和机器码可能不同，安装器会拒绝安装，不能强行混用。

本压缩包不含原游戏程序、关卡、音视频或资源文件。

## 推荐安装方法

1. 解压本压缩包到任意目录；
2. 双击 `安装补丁.cmd`；
3. 按提示输入游戏目录，例如：

   ```text
   E:\1937\1937tzb_1229
   ```

4. 安装器会校验 `M1937.exe`，备份目标目录中可能被覆盖的旧兼容文件，然后复制补丁；
5. 安装完成后，双击游戏目录中的 `启动游戏-窗口模式.cmd`。

也可以在命令行指定游戏目录：

```bat
安装补丁.cmd "E:\1937\1937tzb_1229"
```

## 手动安装

如果不使用安装器，把 `payload` 文件夹中的所有文件和 `Shaders` 子目录复制到 `M1937.exe` 所在目录，允许合并文件夹即可。

不要把 `payload` 文件夹本身套进游戏目录；需要复制的是它里面的内容。

## 启动和全屏

- 推荐入口：`启动游戏-窗口模式.cmd`；
- `Alt+Enter`：在窗口和全屏之间切换；
- 全屏会自动读取当前显示器的 Windows 桌面分辨率。本机验证为 1920×1080，系统以后改成其他分辨率也会自动跟随；
- 再按一次 `Alt+Enter` 会恢复原窗口尺寸；
- `Ctrl+Tab`：临时释放被游戏锁定的鼠标；
- `兼容补丁设置.cmd`：打开 cnc-ddraw 设置程序。

游戏原始画面为 4:3。默认保留正确比例，所以在 16:9 屏幕全屏时左右会有黑边，这是为了避免人物和画面被横向拉伸。

## 卸载与恢复

双击压缩包中的 `卸载补丁.cmd`，输入同一个游戏目录。脚本会移除本补丁安装的文件，并恢复首次安装前备份的同名文件。

首次安装备份保存在：

```text
游戏目录\.1937-compat-backup-before-v1.1.0
```

卸载脚本会保留这个备份目录，避免误删恢复材料。

命令行使用方式：

```bat
卸载补丁.cmd "E:\1937\1937tzb_1229"
```

## 常见问题

### 1. 为什么跳过了开场动画？

两个启动 `.svt` 影片的旧播放循环会在现代 Windows 上长时间不处理窗口消息，是启动假死的主要来源。补丁只跳过两个启动影片，不影响任务和关卡视频。

### 2. VSync 会不会影响操作？

VSync 理论上会增加最多约一帧输入等待，但本游戏属于即时战术游戏。实测开启后桌面合成丢帧率从 18.81% 降到 0%，平滑度收益更明显。

### 3. 全屏有黑边怎么办？

这是 4:3 原始画面在 16:9 显示器上的正确显示。若坚持拉满屏幕，可在设置程序中关闭“保持纵横比”，但画面会变宽，不推荐。

### 4. 安装器提示主程序版本不匹配

不要跳过校验。本补丁包含针对特定 `M1937.exe` 的内存修复，其他版本需要重新定位和测试。

### 5. 杀毒软件提示未知 DLL

`dinput.dll` 是本项目从源码构建的本地兼容层，没有数字签名。源码位于本包 `source\dinput-proxy`，便于审查和重新构建。测试机上的 Windows Defender 扫描未发现威胁。不要因此给整个游戏目录设置永久杀毒排除项。

### 6. 仍在切换关卡时短暂停顿

大型关卡资源从机械硬盘首次读取时可能出现正常加载等待。若只发生在关卡切换，可把完整游戏目录复制到 SSD；不要只移动个别资源文件。
