# 《1937特种兵：敌后武工队》Win10/Win11 兼容补丁

本项目整理并发布《1937特种兵：敌后武工队》绿色版在 Windows 10/11 上的兼容性修复方案。

项目只发布兼容层、补丁包、源码、测试工具和文档，不包含原游戏主程序、关卡、GFL/VWF/SVT/SAV 资源或商业发行内容。使用者需要自行准备合法的绿色版游戏目录。

## 快速使用

下载 [完整补丁包](release/1937特种兵-Win10-Win11兼容补丁-v1.1.0-20260718.zip)，解压后运行 `安装补丁.cmd`，输入 `M1937.exe` 所在目录即可。安装器会校验主程序 SHA-256，并在覆盖前建立恢复备份。

安装后建议运行游戏目录中的 `启动游戏-窗口模式.cmd`。按 `Alt+Enter` 可以切换全屏；全屏会自动采用当前 Windows 桌面分辨率，例如 1920×1080。

## 最终修复内容

- cnc-ddraw 7.1.0.0 将旧 DirectDraw 路径转换到 Direct3D 9；
- 固定 60 FPS、游戏逻辑 Tick 60、VSync，修复桌面合成丢帧造成的微卡；
- 自研 DirectInput 代理补充窗口消息泵，解决启动、菜单和任务阶段“未响应”；
- 对指定版本进行安全的内存补丁，消除资源库误报并跳过两个问题启动影片；
- `Alt+Enter` 动态采用当前主显示器分辨率，同时保持 4:3 原始画面比例。
