# 《1937特种兵：敌后武工队》保存与复刻项目

这个仓库现在包含两个相互独立、可以分别使用的子项目：

- [Patch](Patch/)：让 2001 年原版在 Windows 10/11 上流畅运行的兼容补丁、源码、文档和测试资料。
- [Remake](Remake/)：基于 Godot 4.7 的非官方复刻工程，以及用于解析合法原版资源的 .NET 10 工具链。

## 当前进度

兼容补丁已经完成并可分发。复刻项目处于基础设施和资源格式研究阶段，当前目标是：

1. 从用户本机的合法原版目录生成资源清单；
2. 解析 `GFL/VWF/DBL/SLF` 等 Intuition Engine 私有格式；
3. 将资源转换为 PNG、WAV/OGG 和版本化 JSON；
4. 在 Godot 中完成一个可重复测试的关卡垂直切片；
5. 逐步复原角色、战斗、AI、任务脚本和十二个正式关卡。

## 资产与版权边界

仓库不包含原版游戏的可执行文件、美术、音频、地图、关卡、剧情文本或其他商业发行内容。`Remake` 采用“工程源码 + 本地资源导入器”的模式：使用者需要自行选择合法持有的原版目录，转换产物只保存在被 Git 忽略的 `Remake/LocalAssets/` 中。

复刻工程不是原开发商或发行商的官方项目。“可以在本机解析”不等于“可以公开再分发”；在获得明确授权前，原版资源不会进入 GitHub 提交、Release 或 CI 构建产物。

## 仓库结构

```text
Patch/                  原版 Win10/Win11 兼容补丁子项目
Remake/game/            Godot 4.7 游戏工程
Remake/tools/           .NET 10 资源解析与校验工具
Remake/docs/            架构、格式研究和开发路线
Remake/LocalAssets/     本地导入结果，不进入 Git
```

从 [Remake 使用说明](Remake/README.md) 开始运行复刻原型和资源探针。
