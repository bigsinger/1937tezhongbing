# 《1937特种兵：敌后武工队》保存与复刻项目

本仓库包含两个彼此独立的子项目：

- [Patch](Patch/)：让 2001 年原版在 Windows 10/11 上流畅运行的兼容补丁、源码、文档和测试资料；
- [Remake](Remake/)：使用 Godot 4.7 与 .NET 10 从零实现的现代复刻工程。

## 当前状态

兼容补丁已经完成并可分发。复刻工程已经建立完整的本地资源转换与数据恢复基线：

- 十二个正式关卡 `m000`—`m011` 均可转换为地形 PNG 和版本化 JSON，共 19,199 条场景实体记录；
- 980 个 SPR 的 2,775 个动画组、11,898 帧均已转换，动作语义采用 `20 × 9` 模型，`serial = action × 9 + direction`；
- Godot 的通用动画加载器可以读取任一已知非保留动作，玩家与敌人已按原版逐组 tick 播放移动/站立序列；射击、投掷、近战和死亡的命中/受击过渡仍待接入；
- VWF L1 可见地形、L2 视线/射击遮挡、L3 移动障碍、L4 事件和 L5 手工修正语义已经确认；十二关均生成独立导航数据，玩家和敌人共享防斜穿、动态占位、目标预留和阻挡重规划的八方向 A*；
- 已导出阵营、方向、生命值、默认武器和特殊感知；516 条非空原巡逻路线已接入，敌人可巡逻、发现/追击玩家、按武器范围进入攻击态，并在丢失目标后搜索最后位置；
- 十二关的任务标题、目标依赖、关卡锚点、计数条件、限时和失败条件已经整理为数据驱动任务图；
- 第十二关 `m011` 已识别原版“爆破目标指针被反复覆盖、实际只检查最后一个目标”的缺陷，复刻数据按任务简报恢复为六个爆破目标。

这还不是完整可玩的复刻版。当前可浏览十二关、选择队员、自动绕障碍移动，并观察敌人沿原路线巡逻、发现、追击和进入攻击态；伤害/弹药、警报传播、物品与场景交互、任务世界事件、存档和剧情演出仍在开发。详见 [开发路线图](Remake/docs/ROADMAP.md)、[导航、动态占位与敌人战术](Remake/docs/NAVIGATION_AND_COMBAT.md) 和 [任务恢复说明](Remake/docs/MISSION_RECOVERY.md)。

## 快速开始

需要 .NET 10 SDK、Godot 4.7.1 Standard，以及一份与已知版本匹配的原版目录：

```powershell
cd .\Remake
.\tools\Import-OriginalAssets.cmd "E:\1937\1937tzb_1229"
godot --path .\game
```

导入后可用 `PageUp` / `PageDown` 切换十二关，也可直接指定关卡：

```powershell
godot --path .\game -- --level=m011
```

不导入原版资源也可以启动，工程会显示程序化占位场景。完整的构建、导入和验证命令见 [Remake 使用说明](Remake/README.md)。

## 资产收录策略

仓库不直接收录体积较大的原版可执行文件和美术、音频、地图资源，而是提供可重复运行的本地导入器。转换产物保存在被 Git 忽略的 `Remake/LocalAssets/` 中，这样可以控制仓库与 Release 体积，并让每次导入都有来源哈希、清单和一致性检查。

这是一项仓库体积与可复现构建策略，不对任何素材的权属、授权或可分发状态作法律结论。公开发布时应由发布者根据自己的素材来源与发布范围作出判断。详见 [资产收录与本地导入策略](Remake/ASSET_POLICY.md)。
